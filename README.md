# Install_python_bash
Автоматизирует установку актуальной версии Python через официальный PPA. 
- Использует проверенный репозиторий "deadsnakes/ppa";
- Устанавливает полный набор пакетов: "python", "venv", "dev";
- Автоматически настраивает "pip";
- Выводит полезную информацию для следующего шага (виртуальные окружения).
